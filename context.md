# Contexto del proyecto

Este proyecto consiste en crear una versión simplificada de la interfaz de Airbnb usando Next.js, React, TypeScript y Tailwind CSS.

La idea principal es construir una experiencia donde una persona pueda buscar alojamientos, ver una lista de resultados y entrar al detalle de una habitación o alojamiento específico.

## Páginas del proyecto

### Página de inicio `/`

La página de inicio será la primera vista que verá el usuario. Va a mostrar una barra de navegación, un buscador, filtros por categoría y una grilla de tarjetas de alojamientos.

Desde esta página el usuario podrá explorar opciones y hacer clic en una tarjeta para ver más información.

### Página de catálogo `/catalog`

La página de catálogo mostrará los resultados de búsqueda. Tendrá una cabecera con la cantidad de resultados, un control para ordenar por precio, una lista de alojamientos y un espacio reservado para el mapa.

En esta vista el usuario podrá comparar alojamientos y ordenarlos de forma ascendente o descendente según el precio.

### Página de detalle `/rooms/[id]`

La página de detalle mostrará la información completa de un alojamiento. Tendrá una galería de imágenes, el título, la ubicación, la valoración, información del anfitrión, una lista de servicios y una tarjeta de reserva.

Desde esta página el usuario podrá revisar los detalles principales del alojamiento y simular una reserva.

## Componentes principales

Los componentes principales del proyecto serán:

* Navbar
* Barra de búsqueda
* Filtros de categorías
* Tarjeta de alojamiento
* Grilla de alojamientos
* Encabezado de resultados
* Placeholder de mapa
* Galería de fotos
* Información del anfitrión
* Lista de servicios
* Tarjeta de reserva
* Botón o enlace para volver

## Usuario

El usuario principal de esta plataforma es una persona que quiere encontrar un alojamiento para un viaje. Su objetivo es buscar opciones, comparar precios, revisar la ubicación y los servicios, y elegir un lugar donde hospedarse.
